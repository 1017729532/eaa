# MT5 Technical Indicators Expert Advisor (EA)

一个综合性的MT5专家顾问(EA)，集成了多种技术指标作为交易过滤器。

## 主要功能

- **RSI (相对强弱指数)**: 动量振荡器，用于识别超买/超卖条件
- **KDJ (随机指标)**: 动量指标，比较收盘价与一段时间内价格范围的关系
- **MACD (指数平滑移动平均线)**: 趋势跟踪动量指标
- **布林带 (Bollinger Bands)**: 使用标准差的波动性指标
- **背离检测**: 高级算法检测价格与指标的背离
- **灵活参数**: 所有指标设置都可以完全配置
- **单次交易策略**: 当所有条件满足时只开启第一笔交易

## 技术指标说明

### 1. RSI (相对强弱指数)
- 周期可调 (默认: 14)
- 超买/超卖水平可配置
- 信号: RSI穿越超卖/超买水平

### 2. KDJ (随机指标)
- %K, %D周期可调
- 平滑因子可配置
- 信号: %K在超卖/超买区域穿越%D

### 3. MACD (指数平滑移动平均线)
- 快线、慢线、信号线周期可调
- 信号: MACD线穿越信号线

### 4. 布林带
- 周期、标准差倍数可调
- 信号: 价格从上下轨反弹

### 5. 背离检测
- 支持多指标背离分析
- 回看周期和最小强度可配置
- 检测牛市/熊市背离

## 安装使用

1. 将所有文件复制到MT5的Experts文件夹
2. 在MetaEditor中编译EA
3. 附加到图表并配置参数
4. EA将在满足条件时自动开仓

## 特色功能

- **模块化设计**: 易于维护和扩展
- **参数验证**: 自动验证输入参数的有效性
- **风险管理**: 内置止损止盈设置
- **详细日志**: 提供完整的信号和交易信息
